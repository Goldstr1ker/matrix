# matrix
Matrix Utils (Python)
Небольшой набор функций для работы с матрицами (вложенные списки) в Python: создание матрицы из нулей, печать, поиск максимума, обмен столбцов, проверка квадратности/симметричности и проверка “магического квадрата”.

Что внутри
create_0_matrix(n: int, m: int) -> list
Создаёт матрицу размера n x m, заполненную нулями.

print_matrix(matrix: list, space: int = 3) -> None
Печатает матрицу в консоль, выравнивая элементы по ширине space.

find_matrix_max(matrix: list) -> dict
Находит максимальный элемент и его индекс.
Возвращает словарь вида: {max_value: [row_index, col_index]}.

change_columns(matrix: list, change1: int, change2: int) -> list
Меняет местами два столбца по индексам change1 и change2 (изменяет исходную матрицу).

is_square(sqr_matrix: list) -> bool
Проверяет, является ли матрица квадратной.

is_simmetrical(sqr_matrix: list) -> bool
Проверяет симметричность относительно главной диагонали (только для квадратных матриц).

is_magic_square(matrix: list) -> bool
Проверяет, является ли матрица магическим квадратом:
суммы по всем строкам, столбцам и диагоналям равны; также проверяется наличие всех чисел от 1 до n^2.
